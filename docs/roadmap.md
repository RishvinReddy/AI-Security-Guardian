# Project Roadmap

## Phase 1: Core Scanner & AI Analysis ✅ (v1.0.0)
- 150+ node n8n Mega Workflow
- Deterministic Risk Engine & AI Security Analyst integration
- PostgreSQL 6-Table Storage Schema for evidence retention
- YAML generator for compiling architecture logic

## Phase 2: Intelligence & Trends 🚧 (v1.1.0)
- **Historical Comparison**: Detect newly opened ports or resolved vulnerabilities by comparing SQL state.
- **Threat Intel Aggregation**: Caching API responses in Postgres to minimize rate limiting.
- **Executive Dashboards**: Read-only visualization via Grafana or Metabase.

## Phase 3: SOC Automation 📅 (v2.0.0)
- **Ticketing**: Native Jira and GitHub Issues integration for high-risk findings.
- **Cloud Audits**: Integration with Prowler to assess AWS/GCP/Azure environments.
- **Compliance Scoring**: Automated mapping of findings to SOC2 and CIS frameworks.
