# Contributing to AI Security Guardian

First off, thank you for considering contributing to AI Security Guardian!

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature (`git checkout -b feature/amazing-feature`).
3. If making changes to the Mega Workflow, please modify `generator/workflow-spec.yaml` and re-run `build.py`. **Do not edit the JSON directly.**
4. Commit your changes (`git commit -m 'Add some amazing feature'`).
5. Push to the branch (`git push origin feature/amazing-feature`).
6. Open a Pull Request.

## Workflow Modifications
Because the `AI_Security_Guardian_Mega.json` is massive, manual diffs are nearly impossible. Always rely on the YAML specification.
